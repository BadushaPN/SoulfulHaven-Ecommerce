**Soulful Haven** is a beautiful, highly-responsive Flutter e-commerce application tailored for spiritual and value-based toys, such as Mantra Chanting Baby Plushies (Ganesha, Krishna, Hanuman, etc.).

Designed to be a showcase of modern UI/UX principles, state management, and cross-platform capabilities, this project serves as a comprehensive example of production-ready Flutter development.

## 🚀 Key Features

*   **Breathtaking UI/UX:** A soothing color palette (Teal, Coral, Mint) paired with fluid animations (`AnimatedScale`, `AnimatedContainer`) for a premium shopping experience.
*   **State Management with GetX:** Efficient, reactive state management, dependency injection, and effortless route handling.
*   **Responsive & Adaptive Design:** Custom layouts that adapt flawlessly from mobile screens to wider web/tablet displays.
*   **Dynamic Carousels & Grid Layouts:** Engaging hero sections, auto-scrolling image carousels, and intricate responsive grids for product showcases.
*   **Modern Interactive Components:** Features sticky headers, hover states, interactive product cards, and embedded video presentation sections mimicking modern social commerce (like YouTube Shorts carousels).

## 🛠️ Tech Stack & Libraries

*   **Framework:** [Flutter](https://flutter.dev/) (v3.10+)
*   **Language:** [Dart](https://dart.dev/)
*   **State Management & Routing:** [GetX](https://pub.dev/packages/get)
*   **Typography:** [Google Fonts](https://pub.dev/packages/google_fonts)
*   **Icons:** Cupertino Icons, Material Icons

## 📂 Project Architecture (Feature-First approach)

```text
lib/
├── controller/         # GetX controllers (Business Logic)
├── routing/            # App pages and named routes definitions
├── view/               # UI Screens
│   ├── widgets/        # Screen-specific widgets (e.g., product_card.dart)
│   ├── home_view.dart
│   └── divine_plushies_view.dart
├── widgets/            # Reusable, shared global components (Navbar, Footer, etc.)
└── main.dart           # Application entry point
```

## ⚙️ Getting Started

To run this project locally, ensure you have the [Flutter SDK](https://flutter.dev/docs/get-started/install) installed.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/BadushaPN/<your-repo-name>.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd <your-repo-name>
   ```
3. **Install dependencies:**
   ```bash
   flutter pub get
   ```
4. **Run the app:**
   ```bash
   flutter run
   ```
